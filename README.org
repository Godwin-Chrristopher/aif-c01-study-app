* Overview

This repository now includes a browser-based study app for the AWS Certified AI
Practitioner (AIF-C01) exam.

The app includes:
- Flashcards across focused study decks
- Practice questions by domain
- A hard-question session for mock exam review
- A compact cheat sheet for last-minute revision

[[file:resources/test-image-640x.png]]

* Run Locally

The app is static HTML, CSS, and JavaScript.

Open this file in your browser:

#+BEGIN_SRC text
web/index.html
#+END_SRC

You can also serve it locally if you have Python installed:

#+BEGIN_SRC shell
python -m http.server 8080
#+END_SRC

Then open:

#+BEGIN_SRC text
http://localhost:8080/web/
#+END_SRC

* App Files

- =web/index.html= - main app shell
- =web/styles.css= - styling
- =web/app.js= - app logic
- =web/questions.js= - regular question bank
- =web/hard-questions.js= - hard question session
- =web/hard-questions-part2.js= - hard question session, part 2
- =web/hard-questions-part3.js= - hard question session, part 3
- =web/flashcards.js= - flashcard decks

* Current Content

- 10 flashcard decks
- 52 flashcards
- 75 regular questions
- 150 hard-question entries
- duplicate filtering enabled for the hard-question session

* GitHub Pages

This app can be hosted on GitHub Pages.

The simplest approach is:

1. Move =web/= to =docs/= if you want GitHub Pages to publish directly from the
   repository branch settings.
2. Push the repository to GitHub.
3. In GitHub, open =Settings -> Pages=.
4. Choose =Deploy from a branch=.
5. Select the =main= branch and the =/docs= folder.

If you keep the app in =web/=, GitHub Pages may not offer that folder directly,
depending on the Pages setup you use.
