#+TITLE: AIF-C01 Study Web App
#+AUTHOR: Advina

* Overview

This repository now includes a browser-based study app for the AWS Certified AI
Practitioner (AIF-C01) exam.

The app includes:
- Flashcards across focused study decks
- Practice questions by domain
- A hard-question session for mock exam review
- A compact cheat sheet for last-minute revision

* Run Locally

The app is static HTML, CSS, and JavaScript.

Open this file in your browser:

#+BEGIN_SRC text
docs/index.html
#+END_SRC

You can also serve it locally if you have Python installed:

#+BEGIN_SRC shell
python -m http.server 8080
#+END_SRC

Then open:

#+BEGIN_SRC text
http://localhost:8080/docs/
#+END_SRC

* App Files

- =docs/index.html= - GitHub Pages app shell
- =docs/styles.css= - styling
- =docs/app.js= - app logic
- =docs/questions.js= - regular question bank
- =docs/hard-questions.js= - hard question session
- =docs/hard-questions-part2.js= - hard question session, part 2
- =docs/hard-questions-part3.js= - hard question session, part 3
- =docs/flashcards.js= - flashcard decks

* Current Content

- 10 flashcard decks
- 52 flashcards
- 75 regular questions
- 150 hard-question entries
- duplicate filtering enabled for the hard-question session

* GitHub Pages

This app can be hosted on GitHub Pages.

The repo now includes a =docs/= copy that is ready for GitHub Pages.

1. Push the repository to GitHub.
2. In GitHub, open =Settings -> Pages=.
3. Choose =Deploy from a branch=.
4. Select the =main= branch and the =/docs= folder.
